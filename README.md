# AI180 Project 1 — Cognitive Bias Artifact
**CJ Snyder | Game Design | AI 180**

## Project Overview
An exercise in self-observation across two phases:
- **Phase 1:** Investigate cognitive biases that shape creative work and produce an artifact that makes those biases visible.
- **Phase 2:** Turn the same lens on past AI (vibe coding) sessions and map where biases appeared.

## Biases Documented
| Bias | Context |
|------|---------|
| Einstellung Effect | Constrained AI to existing coding style before exploring alternatives |
| Confirmation Bias | Accepted two different AI fixes without questioning their validity |
| Automation Bias | Delegated verification back to the same system that caused the issue |
| Anchoring Effect | Started AI session without a defined goal; first output became the reference point |
